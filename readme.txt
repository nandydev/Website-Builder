== Description ==

**AI Website Builder by Gravity Write** is a revolutionary WordPress plugin designed to simplify website creation. Ideal for small business owners, bloggers, and designers, this plugin allows users to build professional, responsive websites without any coding skills.

Powered by AI and integrated seamlessly with **Elementor**, the plugin offers intelligent layout suggestions and a library of pre-designed templates. Users can easily customize their designs, update branding elements, and preview changes in real-time before publishing their site.

### Key Features

- **AI-Driven Design**: Generate layouts and content tailored to your needs.
- **Elementor Compatibility**: Easily customize templates with Elementor's user-friendly tools.
- **Live Preview**: Make changes and visualize them instantly.
- **Brand Personalization**: Adjust logos, colors, fonts, and contact details to fit your brand.
- **Extensive Template Library**: Choose from industry-specific pre-built templates.
- **Mobile-Responsive**: All designs are optimized for any device.
- **SEO-Friendly**: Automatically generates websites with SEO best practices in mind.

== Installation ==

1. **Download the Plugin**: Get the ZIP file from WordPress.org or another trusted source.
2. **Upload via WordPress**:
   - Go to your WordPress Dashboard.
   - Navigate to `Plugins > Add New`.
   - Click `Upload Plugin` and choose the downloaded file.
3. **Install and Activate**: Once uploaded, click `Install Now` and then `Activate`.
4. **Start Building**: Find AI Website Builder under the **Gravity Write** menu in your WordPress Dashboard.
